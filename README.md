# Bobby Zhao Sheng Lo - Online Portfolio & CV

This repository contains the source code for the personal portfolio website and online CV of Bobby Zhao Sheng Lo, MD, PhD.

## Description

This is a premium, static portfolio designed to showcase Dr. Lo's professional profile, including his education, awards, research projects, and a comprehensive list of his peer-reviewed publications. The website features a responsive CSS grid layout, modern glassmorphism aesthetics, and extensive SEO optimizations.

## Features

*   **Responsive Portfolio:** Beautiful, structured UI utilizing CSS Grid, designed to look perfect on desktops, tablets, and mobile devices.
*   **Publications Page:** A complete list of publications automatically generated from OpenAlex.
*   **Automated Highlights:** The publications generator automatically surfaces high-impact papers (e.g., published in Nature, Gastroenterology) and highly cited papers into a dedicated "Highlights" section, abbreviating the author list to focus on Dr. Lo.
*   **Live Rolling Banner:** An animated marquee actively showcasing the latest publications across the site.
*   **Aggressive SEO:** Advanced JSON-LD markup, expanded Open Graph tags, and crawler-friendly structural blocks ensure high visibility on Google for IBD research in Denmark and GAIN.
*   **Weekly Auto-Updates:** A GitHub Action is set up to automatically run the Python scraper, generate new HTML/SEO content, and push changes to the repository every week.

## Setup

The website is built with plain HTML, semantic CSS, and JavaScript. The underlying data and SEO content are generated by a Python script.

To run the project locally, you will need Python installed.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Bobby-zs-lo/Bobby-zs-lo.github.io.git
    cd Bobby-zs-lo.github.io
    ```
2.  **Install Python dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
3.  **Generate the publications list and SEO content:**
    ```bash
    python generate_publications.py
    ```
4.  **Open `index.html` in your browser to view the website.**

## Automated Publication Updates

A GitHub Action is configured to run the `generate_publications.py` script once a week (every Sunday at midnight). This script fetches the latest publications from Dr. Lo's OpenAlex profile, extracts citation data and journal metrics to build an automatic "Highlights" section, generates the rolling marquee, and updates the `publications.html` and `index.html` files.

The workflow is defined in `.github/workflows/update-publications.yml`. Any changes are automatically committed to the `main` branch.
